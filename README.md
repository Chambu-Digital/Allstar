# Allstar Tech

## Overview

Allstar Tech is a modern, responsive e-commerce platform specializing in computers and electronic products. We provide customers with an intuitive shopping experience to explore cutting-edge technology, compare products, and make informed purchasing decisions.

This project focuses on clean UI, performance, and scalability, making it suitable for both business presentation and future expansion (advanced e-commerce features, dashboards, integrations).

## Features

- Responsive and mobile-friendly design
- Modern teal blue and orange color theme
- Comprehensive product catalog for computers and electronics
- Advanced search and filtering capabilities
- User authentication and account management
- Shopping cart with wholesale pricing support
- Product reviews and ratings
- Contact and inquiry functionality
- SEO-friendly structure
- Fast loading and optimized assets

## Tech Stack

- Language: TypeScript
- Styling: Tailwind CSS
- Frontend Framework: Next.js 16
- Database: MongoDB with Mongoose
- State Management: Zustand
- Authentication: JWT with Google OAuth
- UI Components: Radix UI
- Version Control: Git & GitHub

## Brand Identity

- Primary Color: Teal Blue (#0d9488)
- Secondary Color: Orange (#f97316)
- Focus: Computers, Laptops, Electronics, Accessories
