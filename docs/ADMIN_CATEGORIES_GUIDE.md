# Admin Categories Management Guide

## Overview
The Categories section allows you to organize your technology products into logical groups, making it easier for customers to browse and find what they need.

## Accessing Categories
1. Log in to the admin panel at `/admin/login`
2. Navigate to **Categories** from the sidebar menu
3. You'll see a list of all existing categories

## Creating a New Category

### Step 1: Click "Add Category"
- Click the **"Add Category"** button in the top right corner

### Step 2: Fill in Category Details

#### Required Fields:
1. **Category Name** (Required)
   - Enter a clear, descriptive name
   - Examples: "Laptops", "Desktop Computers", "Gaming Accessories"
   - The system will automatically generate a URL-friendly slug

2. **Description** (Required)
   - Write a brief description of what products belong in this category
   - This helps customers understand what they'll find
   - Example: "High-performance laptops for work, gaming, and creativity"

3. **Category Image** (Required)
   - Upload a representative image for the category
   - Recommended size: 800x600px or larger
   - The image will be watermarked with "© Allstar Tech"
   - Supported formats: JPG, PNG, WebP

4. **Icon** (Required)
   - Select an icon that represents the category
   - Available technology icons:
     - Laptop - For laptop computers
     - Monitor - For displays and screens
     - Cpu - For processors and components
     - HardDrive - For storage devices
     - Smartphone - For mobile devices
     - Tablet - For tablets
     - Keyboard - For input devices
     - Mouse - For pointing devices
     - Headphones - For audio equipment
     - Wifi - For networking equipment
     - Printer - For printers and scanners
     - Camera - For cameras and imaging
     - Gamepad2 - For gaming products
     - Battery - For power supplies and UPS
     - Cable - For cables and adapters
     - Usb - For USB accessories
     - Server - For server equipment
     - Database - For storage solutions

5. **Active Status** (Checkbox)
   - Check to make the category visible on the website
   - Uncheck to hide it temporarily

### Step 3: Save
- Click **"Create Category"** to save
- You'll be redirected to the categories list

## Editing a Category

1. Find the category in the list
2. Click the **Edit** button (pencil icon)
3. Update any fields as needed
4. Click **"Update Category"** to save changes

## Managing Category Visibility

### Toggle Active/Inactive
- Click the **Eye/Eye-Off** icon to quickly toggle visibility
- Active categories appear on the website
- Inactive categories are hidden but not deleted

### Visual Indicators
- **Green badge**: Category is active and visible
- **Red badge**: Category is inactive and hidden

## Deleting a Category

⚠️ **Warning**: Deleting a category is permanent!

1. Click the **Delete** button (trash icon)
2. Confirm the deletion in the popup
3. The category will be permanently removed

**Note**: Before deleting, consider:
- Are there products assigned to this category?
- Would it be better to deactivate instead?

## Best Practices

### Naming Categories
- Use clear, customer-friendly names
- Be specific but not too narrow
- Examples:
  - ✅ "Gaming Laptops"
  - ✅ "Wireless Keyboards"
  - ❌ "Misc Items"
  - ❌ "Other"

### Organizing Categories
- Create 10-15 main categories
- Keep descriptions concise (2-3 sentences)
- Use high-quality images that represent the category well
- Choose icons that are immediately recognizable

## Adding Products to Categories

After creating categories:
1. Go to **Products** section
2. When creating/editing a product, select the appropriate category
3. Products will automatically appear in their assigned category on the website

## Troubleshooting

### Category not showing on website?
- Check if the category is marked as "Active"
- Ensure the category has an image uploaded
- Verify the category has products assigned to it

### Image upload failing?
- Check file size (should be under 5MB)
- Use supported formats: JPG, PNG, WebP
- Try a different browser if issues persist

### Slug conflicts?
- Each category needs a unique slug
- If you get an error, try a slightly different name
- The system auto-generates slugs from the category name

1. **Start with main categories** - Create broad categories first
2. **Use quality images** - Good visuals improve customer experience
3. **Write clear descriptions** - Help customers understand what's in each category
4. **Review regularly** - Update categories as your inventory changes
5. **Test visibility** - Check the website to see how categories appear to customers
